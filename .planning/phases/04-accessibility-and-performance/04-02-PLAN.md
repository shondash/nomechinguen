<!-- Context from research: focus color contrast calculations -->

Focus ring contrast verified by research:
- FOCUS_STYLE_LIGHT: boxShadow "0 0 0 2px #163A5F" — 11.6:1 vs white, 10.7:1 vs #F7F5F0 page bg (PASSES 3:1)
- FOCUS_STYLE_DARK: boxShadow "0 0 0 2px #DAA520" — 5.2:1 vs header #163A5F, 6.7:1 vs install banner #1B2A1B (PASSES 3:1)

Why box-shadow not outline: outline-offset creates a gap showing background color. The contrast is measured against adjacent area. Gold on page bg gap = 2.05:1 (FAIL). box-shadow sits flush = comparison is against element bg.

Focus application pattern (no React state needed):
```javascript
onFocus={e => { e.currentTarget.style.boxShadow = "0 0 0 2px #163A5F"; }}
onBlur={e => { e.currentTarget.style.boxShadow = ""; }}
```

Dark context variant (header tone toggle, download link):
```javascript
onFocus={e => { e.currentTarget.style.boxShadow = "0 0 0 2px #DAA520"; }}
onBlur={e => { e.currentTarget.style.boxShadow = ""; }}
```
