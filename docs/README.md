## Como usar esta documentacion

- **Trabajadores:** Lee los documentos 01-07 para conocer tus derechos basicos.
- **Desarrolladores:** Consulta el documento 13 (formulas) antes de modificar cualquier calculo.
- **Actualizacion anual:** Cada enero, actualiza el documento 11 (valores monetarios) con los nuevos SM y UMA publicados por CONASAMI e INEGI.
